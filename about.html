About unownx

unownx is a private searx instance. It is hosted with heroku and resets every day (deleting all temporary data with it) for server health.

searx is a metadata search engine. It is intended for privacy, and it collects information from various other search engines (such as Google or DuckDuckGo). searx itself is private, and doesn't record any information about you. If you'd like to learn more, visit the searx website.

Happy searching!
